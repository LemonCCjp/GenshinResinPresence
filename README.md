# GenshinResinPresence

[exeダウンロード](https://github.com/LemonCCjp/GenshinResinPresence/releases/tag/GenshinImpact)

# 1.DiscordApplicationの作成

[DiscordDeveloperPortal](https://discord.com/developers/applications)でアプリケーションを作成し、OAuth2タブからClientIDをコピー

user_data.json内のCLIENT_ID欄に貼り付け

# 2.HoyoLABから認証情報を取得

[HoyoLAB](https://www.hoyolab.com/home)内でデベロッパーツール(F2等)を開き、Cookieから以下の3つの認証情報をコピーし貼り付け

ltmid_v2

ltuid_v2

ltoken_v2

※ltuid_v2とuidは別物です。uid欄には原神ゲーム内で使用されている、8または18から始まるuidを入力してください

# 3. 起動

GenshinResinPresenceを起動し、入力した情報に誤りが無ければDiscordアカウントのアクティビティ欄に樹脂数等の情報が表示されます

# 4.追加設定

`Shortcut`フォルダ内にGenshinImpact.exeのショートカット(.lnk)を配置すると

システムトレイから`GenshinImpactを起動`を押すと原神を起動できるようになります。
